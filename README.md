# Route Matching test project

## Configuration

Set in `appsettings.json` ApiId and ApiCode for here integration.:
```
{
  "Logging": {
    "LogLevel": {
      "Default": "Warning"
    }
  },
  "AllowedHosts": "*",
  "HereApi": {
    "app_id": "FILL",
    "app_code": "FILL"
  }
}
```

## Usage
Build and start, then navigate to `http://localhost:50138/swagger/`
